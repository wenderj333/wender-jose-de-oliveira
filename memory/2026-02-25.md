## Video Display Issue Found

### Problem Identified
- **Uploaded videos not showing correctly in Mural**
- Posts appear as blank/gray boxes instead of video thumbnails
- Root cause: Videos saved with `media_type: 'image'` in database
- Cloudinary URLs contain `/video/` but frontend treats them as images

### Solution in Progress
- Will modify MuralGrid.jsx to auto-detect video URLs from Cloudinary
- Ignore `media_type` field and use URL patterns instead
- If URL contains `/video/` or `resource_type=video` → show as video
- This will recover all "lost" videos without database reprossessing
